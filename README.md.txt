# Development Environment Setup

This project is to set up a basic development environment.

## Tools Installed
* MySQL
* Git

## Folder Structure

* `src`: Contains source code files (like hello.txt).
* `docs`: For documentation.
* `resources`: For any other resources.

## Setup Instructions

1.  Installed each of the tools listed above.
2.  Created a github repository.
3.  Cloned the repository to my local machine.
4.  Created the above folder structure.
5.  Added this README file.
